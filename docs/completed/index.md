# Completed

The **Completed** view shows cards that are finished. By default these come from the active board folder and the archive. Tick **Search all** and click **Search** to also include cold storage, abandoned cards, and in-flight (open) cards.

![Screenshot: The completed view](../screenshots/demo-completed-full.png)

You can filter the card list by owner, swimlane, and **Closed** date (this week, this month, last week, last month, or all). Use **Search** to match title, owner, links, and related fields.

Each row shows a source badge: **Board**, **Archive**, **Cold**, **Abandoned**, or **In-flight**, so you can tell where a card was found.

You can use the **Completed** view to see all closed tasks, while the [Board](../board/index.md) view shows at most five recent cards per Done column.

For aggregate trends across completions, use [Charts](../charts/index.md).
